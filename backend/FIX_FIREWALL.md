# Решение проблемы с Windows Firewall

## Проблема
Windows Firewall блокирует Node.js при попытке скачать Prisma engines, 
потому что сеть определена как "Общедоступная" (Public).

## Решение 1: Изменить тип сети на "Частная" (РЕКОМЕНДУЕТСЯ)

### Способ A: Через GUI (проще всего)

1. Откройте **Параметры Windows** (Win + I)
2. Перейдите в **Сеть и Интернет**
3. Выберите **Ethernet** (или Wi-Fi)
4. Нажмите на активное подключение "Сеть"
5. В разделе **Тип сетевого профиля** выберите **Частный**
6. Закройте окно

### Способ B: Через PowerShell с правами администратора

1. Запустите PowerShell **от имени администратора** (правой кнопкой → Запуск от имени администратора)
2. Выполните:

```powershell
Set-NetConnectionProfile -Name "Сеть" -NetworkCategory Private
```

3. Проверьте изменение:

```powershell
Get-NetConnectionProfile
```

## Решение 2: Добавить Node.js в исключения брандмауэра

Если не хотите менять тип сети, добавьте Node.js в разрешённые программы:

### Через PowerShell (от администратора):

```powershell
# Найти путь к node.exe
$nodePath = (Get-Command node).Path

# Добавить правило для исходящих соединений
New-NetFirewallRule -DisplayName "Node.js Outbound" -Direction Outbound -Program $nodePath -Action Allow

# Добавить правило для входящих соединений (опционально)
New-NetFirewallRule -DisplayName "Node.js Inbound" -Direction Inbound -Program $nodePath -Action Allow
```

### Через GUI:

1. Откройте **Панель управления** → **Система и безопасность** → **Брандмауэр Защитника Windows**
2. Слева: **Разрешение взаимодействия с приложением или компонентом**
3. Нажмите **Изменить параметры**
4. Нажмите **Разрешить другое приложение**
5. Найдите `node.exe` (обычно в `C:\Program Files\nodejs\node.exe`)
6. Добавьте, поставьте галочки для **Частная** и **Публичная**
7. ОК

---

## Решение 3: Временно отключить брандмауэр для общедоступной сети

⚠️ **НЕ РЕКОМЕНДУЕТСЯ** для постоянного использования!

1. Откройте **Брандмауэр Защитника Windows**
2. Слева: **Включение и отключение брандмауэра**
3. Для **Параметры для общедоступной сети**: выберите **Отключить**
4. После установки Prisma включите обратно!
