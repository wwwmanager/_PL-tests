# P0-B: WB-BLANK-RESERVE-002 - Резервирование Бланка

## Изменённые файлы

- `components/waybills/WaybillDetail.tsx` (строки 100-110)

## Что изменено

1. Убрана автовыборка первого бланка в UI (строки 103-109)
2. Backend резервирует бланк при сохранении через `reserveNextBlankForDriver`
3. Номер бланка возвращается с backend после создания

## Как проверить

1. Создать DRAFT1 → сохранить → должен получить ЧБ 000001
2. Создать DRAFT2 → сохранить → должен получить ЧБ 000002
3. Проверить что DRAFT1 держит 000001 (не освобождается)
