# P0-A: WB-NUM-001 - Исправление Формата Номера ПЛ

## Изменённые файлы

- `components/waybills/WaybillDetail.tsx` (строка 1471)

## Что изменено

1. Замена отображения бланка в селекте с `{b.series} {b.number}` на `{b.formattedNumber}`
2. Теперь показывается "ЧБ 1" вместо "ЧБ 000001"

## Как проверить

1. Открыть создание нового ПЛ
2. Выбрать водителя (появится список бланков)
3. Проверить dropdown - должен показывать "ЧБ 1" вместо "ЧБ 000001"
